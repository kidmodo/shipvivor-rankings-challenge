    Scoring Rules

    Shipvivor Rankings Challenge: Rules + Features Overview

    Season Structure

    • Week 1 is excluded from scoring.
    • Week 2 is a legacy scoring week and is frozen as stored data (not recalculated).
    • Weeks 3 through 12 use scaled scoring based on the remaining cast size.

    Scoring (Week 3 Onward)

    • Each elimination uses your rank position for that castaway in that week's lineup.
    • Let N be the number of active castaways in that week and r your rank for the eliminated player (clamped to 1..N).
    • Cap formula: cap = 22 * (N / 22)^0.7
    • Per elimination: pointsRaw = 1 + (r - 1) * (cap - 1) / (N - 1)
    • Points are rounded to one decimal place.
    • If N = 1, the elimination awards 1.0 point.
    • For double boots or multi-elimination episodes, elimination points are averaged so one episode yields one weekly score swing.

    Legacy Week 2 Handling

    • Week 2 scores remain exactly as stored in the database.
    • Those values are included in totals as legacy data.

    Editing + Locking

    • Only the active week lineup can be edited.
    • Lineups lock at episode start: Wednesday 8:00 PM ET (from Week 3 onward).
    • The countdown card shows time remaining until lock.
    • After lock, active week lineup is read-only.

    Skip + Omit Rules

    • Players can skip the active week (scores 0.0 for that week).
    • Admin can omit a player from a completed week (scores 0.0 for that week).
    • Late joiners are treated as omitted for weeks before they joined.

    Tabs + Views

    • Set Lineup: set active-week order, notes, winner picks, and save lineup.
    • Standings: full points table by week. Active week shows a check if lineup is saved, or no-lineup icon if not saved yet.
    • Others Rankings: completed-week report table plus week/user lookup to view other players' rankings and notes.
    • Smack Talk: shared chat with selectable castaway avatar.

    Admin Controls

    • Advance week, mark voted-off castaways, edit tribe history, manage users, and update recap.
    • Admin can mark a user note as Comment of the Week for a completed week.
    • Comment of the Week displays on the following week's recap card.
